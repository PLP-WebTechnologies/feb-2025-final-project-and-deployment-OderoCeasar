Understanding JavaScript Variables

Date: May 13, 2025 | Author: Ceasar

What is a Variable?

A variable in JavaScript is a container for storing data values. You can think of it like a labeled box in which you place information.

Types of Variables

There are three main ways to declare variables:

  • var – old way (avoid using in modern JS)
  • let – block-scoped, preferred for changeable data
  • const – block-scoped, used for constants

let name = "Alice";
const age = 25;
var message = "Hello";
